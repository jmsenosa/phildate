# phildate
convert numbers to Philippine contact number version 



```
$(".philcontact").philcontact({ 
    countryCode : "+63", 
    maxNumber   : 10,
    minNumber   : 7,
    inputClass  : "philcontact-input",
    customClass : ""
});
```
